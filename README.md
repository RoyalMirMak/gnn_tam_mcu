# Разработка и реализация алгоритмов графовых нейронных сетей для микроконтроллерных систем

Данный репозиторий содержит исходный код, прилагаемый к курсовой работе на тему "Разработка и реализация алгоритмов графовых нейронных сетей для микроконтроллерных систем" студента 3-го курса ПМИ ФКН ВШЭ Махлина Мирона.

## Структура репозитория

- Папка `Core` является содержательной частью проекта для STM32CubeIDE и содержит код для загрузки на микроконтроллер и дальнейшего запуска модели на нём.

- Файл `benchmark_cool.py` содержит код, использовавашийся для бенчмарка модели, запущенной на устройстве с ограниченными ресурсами: измерения времени и метрик качества.

- Файл `rename_model.py` является вспомогательным скриптом для удобного переключения между моделями в рамках одного проекта в IDE.

- Подмодуль `gnn-tam` является форком официального репозитория статьи "[Graph Neural Networks with Trainable Adjacency Matrices for Fault Diagnosis on Multivariate Sensor Data](https://doi.org/10.1109/ACCESS.2024.3481331)", доработанным в соответствии с описанными в работе модификациями. В частности:

    - Новый файл `fold_gnn_tam.py` предназначен для трансформации обученной модели формата PyTorch в модель ONNX, готовую для запуска на микроконтроллере.

    - Новый файл `quantize.py` используется для статической и динамической квантизации модели.

    - Новый файл `cnn.py` содержит имплементацию базовой архитектуры свёрточной нейронной сети, использовавшейся как эталон для сравнения.
